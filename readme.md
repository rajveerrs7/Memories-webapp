# Memories WebApp

Memories WebApp is a simple CRUD (Create, Read, Update, Delete) application where users can add and manage pictures along with their title, description, and tags. Built using the MERN stack (MongoDB, Express, React, Node.js), it provides a seamless experience for storing and sharing your memories.

## Features

- **Add Memories**: Upload images with a title, description, and relevant tags.
- **View Memories**: Browse through all the memories created by users.
- **Edit Memories**: Update details and images for your memories.
- **Delete Memories**: Remove memories that you no longer want to keep.

## Tech Stack

- **MongoDB**: Database for storing memory data and user information.
- **Express**: Backend web framework for building the API.
- **React**: Frontend library for crafting an interactive user interface.
- **Node.js**: JavaScript runtime for backend development.
